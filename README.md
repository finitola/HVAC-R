# HVAC/R Knowledge Base ❄️🔥

**საგანმანათლებლო და პრაქტიკული პლატფორმა HVAC (გათბობა, ვენტილაცია, კონდიცირება) ინჟინრებისთვის.**

ეს არის **Single Page Application (SPA)**, რომელიც შექმნილია სუფთა (Vanilla) JavaScript-ზე, გარე ბიბლიოთეკების და ფრეიმვორკების გარეშე. საიტი არის სრულად ოპტიმიზირებული მობილური მოწყობილობებისთვის და მუშაობს როგორც აპლიკაცია (PWA).

---

## 🚀 ფუნქციონალი

აპლიკაცია მოიცავს შემდეგ განყოფილებებს:

- **📚 სწავლა:** დეტალური ინფორმაცია სისტემებზე, კომპონენტებსა და მაცივარაგენტებზე (ფრეონებზე).
- **⚡ ელექტრონიკა:** ელექტროობის საფუძვლები HVAC სისტემებში.
- **📝 ტესტირება:**
  - სასწავლო რეჟიმი (კითხვა-პასუხი).
  - საგამოცდო სიმულაცია (დროზე და ქულებზე).
- **🧮 კალკულატორები:** პროფესიული ხელსაწყოები ინჟინრული გათვლებისთვის.
- **🛠 სერვისი:** პრაქტიკული რჩევები და სერვისის ინსტრუქციები.

## 🛠 გამოყენებული ტექნოლოგიები

- **Frontend:** HTML5, CSS3, JavaScript (ES6+ Modules).
- **Architecture:** SPA (Single Page Application) - მუშაობს გვერდის გადატვირთვის გარეშე.
- **Routing:** Custom Hash Router (საკუთარი დაწერილი როუტერი).
- **Mobile Experience:** PWA (Progressive Web App) მხარდაჭერა - შესაძლებელია მობილურში აპლიკაციად დაყენება (Add to Home Screen).
- **Hosting:** GitHub Pages.

## 📱 როგორ დავაყენოთ მობილურზე?

საიტი იყენებს **PWA** ტექნოლოგიას.

1. გახსენით საიტი Chrome-ში (Android) ან Safari-ში (iOS).
2. Android: დაელოდეთ შეტყობინებას "Add to Home Screen" ან მენიუდან აირჩიეთ "Install App".
3. iOS: დააჭირეთ "Share" ღილაკს და აირჩიეთ "Add to Home Screen".
4. აპლიკაცია გამოჩნდება თქვენს ეკრანზე ლურჯი ლოგოთი და გაიხსნება სრული ეკრანის რეჟიმში.
