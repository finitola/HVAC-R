* პასუხი გაგცეთ **48 საათის განმავლობაში**

### 🔒 Security Considerations

პლატფორმის არქიტექტურის გათვალისწინებით:

* აპლიკაცია არის **Client-side only SPA**
* არ გამოიყენება:

  * Backend სერვერი
  * მონაცემთა ბაზა
  * ავტორიზაცია / ავთენტიფიკაცია
* არ ინახება:

  * პერსონალური მონაცემები
  * მომხმარებლის IP მისამართები
  * Cookies (თუ დამატებით არ იქნება ჩართული მესამე მხარის სერვისი)

შესაბამისად, უსაფრთხოების ძირითადი რისკები შეიძლება მოიცავდეს:

* XSS (Cross-Site Scripting)
* არასანდო გარე სკრიპტების ჩართვა
* ბრაუზერის ლოკალურ მონაცემებზე (localStorage) არასწორი წვდომა

### 🛡 Best Practices Used

* Vanilla JavaScript (გარე ბიბლიოთეკების გარეშე)
* Inline script-ების მინიმალური გამოყენება
* Responsive და Mobile-first დიზაინი
* Hosting GitHub Pages-ზე (HTTPS by default)
